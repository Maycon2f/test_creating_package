# matematica

Description. 
The package matematica is used to:
	- to add (a + b)
	- to subtract (a - b)

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install package_name

```bash
pip install matematica
```

## Usage

```python
from matematica import soma, subtracao
soma()
subtracao()
```
